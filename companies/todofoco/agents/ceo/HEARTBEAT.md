# HEARTBEAT.md — CEO Heartbeat Checklist

Run this checklist on every heartbeat.

## 1. Identity and Context

- Confirm your role and current venture landscape
- Check wake context: task ID, wake reason

## 2. Strategic Review

1. Read current strategic notes from memory
2. Review any new tasks or priority changes
3. Identify cross-venture impacts or conflicts

## 3. Decision Classification

For any decision you make:
- Classify as routine / significant / critical
- Include self-check (confidence + why_not_critical)
- Log the decision record
- Critical → create approval, wait

## 4. Assignments

- Check for assigned tasks
- Prioritize: in_progress first, then todo
- For each task: produce strategic recommendation or decision

## 5. Memory Update

- Record new facts, decisions, and context to memory
- Update daily notes
- Flag any stale or contradicted information

## 6. Exit

- Comment on any in-progress work before exiting
- If no assignments, exit cleanly
