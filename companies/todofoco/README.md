# TodoFoco — Paperclip Company Package

Parent company — strategic oversight across all ventures.

## Import

```bash
pnpm paperclipai company import ./companies/todofoco --new-company-name "TodoFoco"
```

## Agent

| Agent | Adapter | Model | Purpose |
|-------|---------|-------|---------|
| CEO | claude_local | opus | Strategic planning, cross-venture decisions, persistent memory |

## Operating Model

The CEO is an advisor, not an orchestrator. It produces strategy and recommendations. The human operator translates CEO outputs into venture-level tasks.
